## Installation
The project uses ES6 Javascript and ThreeJS.

To install, first clone this directory. Then cd into the directory and run:

```
npm install .
```

## How to Run

To start the server, run:
```
node server.js
```

To run the web app locally, run:
```
python -m http.server
```
